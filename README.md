# Projektna naloga

## Opis projekta
Projekt obravnava razvoj sistema za prepoznavanje gest z uporabo Time-of-Flight (ToF) senzorja in RGB kamere. Sistem združuje globinske in slikovne podatke ter omogoča zaznavanje gibanja in klasifikacijo različnih gest roke.

V okviru projekta so bile implementirane funkcionalnosti zajema podatkov, predobdelave signalov, klasifikacije gest ter integracije vseh komponent v enotno aplikacijo.
